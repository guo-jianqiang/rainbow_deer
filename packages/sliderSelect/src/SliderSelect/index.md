支持滑动、点击的选择器组件

## install

```shell script
npm install @rainbow_deer/slider-select
or
yarn add @rainbow_deer/slider-select
```

## usage

### basic

```tsx
import React from 'react'
import SliderSelect from '@rainbow_deer/slider-select'

export default () => {
  return <SliderSelect width={300} value={1} />
}
```
